# Map traversal

Simple breadth-first search algorithms for 2d grid map.

## Build
```
make
```

## Run
1. Custom settings.
```
./map_traversal
```
2. Using default settings.
```
./map_traversal -d 
```

## Usage
1. Move character.
```
wasdqezc 
```
2. Encrease/Decrease path length.
```
+-
```
3. Quit.
```
r
```
